College Campus Helper (CSC 330 Final Project)
CollegeCampusHelper is a student utility app built using Java and Android Studio.

This project includes:
1.	Home Dashboard – Quick navigation to the app’s features.
2.	Notes Section – Allows students to write, edit, and clear personal notes using persistent storage.
3.	Profile Selection – Allows users to select a profile picture from their device gallery.

Build Requirements:
- Android Studio
- Java 17+
- Android SDK 26+

How to Build the App:
1. Download the program as a ZIP before extracting it.
2. Open Android Studio.
3. Click "Open an Existing Project".
4. Select the folder "CollegeCampusHelper/".
   
How to Run the App:
1.  In Android Studio, open Device Manager.
2.  Create an emulator / Enable Developer Options and USB debugging before plugging in a device.
3.  Click Run.
